## Initial Launch - Language Learning Platform

### Core Features

- **Vocabulary Management** — Add, edit, and organize vocabulary words with translations, examples, and notes
- **Sentence Practice** — Interactive sentence-building activities with audio playback
- **Progress Tracking** — Monitor learning progress with detailed statistics and encoding strength metrics
- **Study Sessions** — Structured study activities with spaced repetition
- **Audio Support** — Text-to-speech integration for pronunciation practice
- **Cross-Platform Sync** — Seamless synchronization between mobile and web applications
- **Multi-Language Support** — Built-in support for Korean language learning with extensible architecture for additional languages
- **Search & Filter** — Powerful search capabilities across vocabulary and activities
- **Data Export** — Export vocabulary and study data for backup and analysis

### Technical Foundation

- **.NET MAUI** — Native mobile experience on iOS, Android, Mac Catalyst
- **Blazor Web** — Full-featured web application
- **.NET Aspire** — Distributed application orchestration
- **Entity Framework Core** — Robust data persistence with SQLite
- **CoreSync** — Real-time multi-device synchronization
