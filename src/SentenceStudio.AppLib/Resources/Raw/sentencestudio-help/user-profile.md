# User Profile

SentenceStudio supports multiple user profiles on a single device. Each profile has its own vocabulary, history, and skill profile. Switching profiles changes the data visible throughout the app.

## Creating a profile

During onboarding you create an initial profile. Additional profiles can be added later from the profile picker. Give each profile a display name and choose the target language you are learning.

## Switching profiles

Open the profile menu and select a different user. The app re-scopes all repositories to the active profile; vocabulary, conversations, and activity history update immediately.

## What is per-profile

- Vocabulary words and lists
- Skill profiles and active selection
- Learning Resources you imported or generated
- Activity history and progress
- Conversation history

## What is shared

- App language and theme preferences
- Authentication tokens for the cloud API
- Secure storage containing API keys

If data looks wrong after switching, verify the active profile in the profile menu before investigating further.
