# Settings

The Settings page controls app-wide preferences that are not tied to a single profile.

## Language

Choose the app UI language. SentenceStudio currently ships English and Korean UIs. Changing the UI language does not change your learning target language — that is part of your user profile.

## Theme

Light, dark, and system-follow are supported. The choice applies across all activities.

## Audio

Audio-related preferences such as playback category and default voice for speech synthesis live here. On macOS and iOS the playback category determines whether audio mixes with other apps or interrupts them.

## Account

- Sign out of the current account.
- View the API endpoint the app is talking to.
- Trigger a manual sync to force reconciliation with the server.

## Developer / diagnostics

Debug Health exposes internal counters useful when filing bug reports. These are additive and do not modify data.
