# Getting Started

SentenceStudio is an AI-assisted language-learning app focused on Korean. It combines structured activities, vocabulary management, and conversational practice backed by live AI models.

## First run

1. Create or log in to a user profile. Multiple profiles are supported so different learners can share a device without mixing progress.
2. Complete onboarding to set your native and target languages and preferred skill levels.
3. From the dashboard, pick an activity or let the daily plan suggest one.

## Core concepts

- **Profile** — your identity inside the app. Vocabulary lists, history, and progress are scoped to the active profile.
- **Skill Profile** — a persona describing what you're learning (topic, level, style). Activities use the active skill profile to tailor AI output.
- **Learning Resource** — a text, dialogue, or corpus you imported or generated. Activities can target a specific resource.
- **Activity** — a timed practice session (Cloze, Writing, Translation, and others). Each activity records what you completed so the dashboard can show progress.

## Where to go next

- Dashboard: see what's due today and jump into the daily plan.
- Vocabulary: review and manage your word lists.
- Settings: change language, theme, and sync behavior.
