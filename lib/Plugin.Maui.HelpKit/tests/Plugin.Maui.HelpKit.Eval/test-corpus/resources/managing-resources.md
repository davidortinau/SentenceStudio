# Learning resources

Learning Resources are source materials (articles, books, videos, scripts) that drive vocabulary and activity content.

## Opening the Resources page

Select Resources from the navigation menu. You see a searchable list of all your resources.

## Adding a resource

Tap "Add Resource" in the header. Fill in the title, type, language, and body content. Save to store it in your library.

## Importing content

For bulk content, use the Import page. You can import text resources from files or paste content to create a resource automatically.

## Filtering

The Resources page supports filtering by type, by language, and by free-text search over titles and bodies.
