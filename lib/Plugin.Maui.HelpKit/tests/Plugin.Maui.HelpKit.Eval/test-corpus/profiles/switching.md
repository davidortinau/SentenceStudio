# User profiles

SentenceStudio supports multiple user profiles so that different learners, or different language pairs for the same learner, can track progress independently.

## Switching profiles

Open the Profile page from the navigation menu. Your current profile information appears at the top. Use the profile switcher to change the active profile.

## Creating a profile

From the Profile page you can create a new profile by entering a name, selecting your native language, selecting the language you are learning, and saving.

## What a profile stores

A profile stores:

- Personal information (name, email)
- Native language and target language
- Progress across every activity
- Vocabulary lists and learning resources assigned to the profile
