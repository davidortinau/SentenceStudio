# Filtering vocabulary

The Vocabulary page supports filtering so you can focus on a subset of your words.

## Opening filters

On a phone screen, tap the funnel icon in the header to open the filter panel. The icon fills in when any filter is active.

## Available filters

- Status: filter by learning status such as new, learning, or mastered.
- Resource: show only words associated with a specific learning resource.
- Search: type in the search field to match term or translation text.

Filters combine — words shown must satisfy every active filter.
