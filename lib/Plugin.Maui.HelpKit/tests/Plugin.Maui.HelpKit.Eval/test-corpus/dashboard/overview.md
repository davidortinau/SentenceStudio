# Dashboard overview

The Dashboard is the landing page when you open SentenceStudio. It is located at the root route `/`.

## What you see

- A header titled "Dashboard" with a refresh button.
- If you are a brand new user, the header reads "Welcome" and you are guided through getting started.
- If your profile has a Today's Plan configured, the dashboard shows the plan for the current day. You can regenerate the plan from the actions menu.
- Cards link into the available activities: Cloze, Writing, Translation, Word Association, Vocabulary Quiz, and Reading.

## Common tasks

- To start any activity, select its card from the dashboard.
- To refresh the dashboard data, tap the circular arrow icon in the toolbar.
- To rebuild Today's Plan, open the actions menu and choose "Regenerate Plan".

## Navigating away

Selecting an activity navigates you into that activity's page. Each activity page has a back arrow that returns you to the dashboard.
