# Writing activity

The Writing activity asks you to compose a sentence in the target language using a given prompt or vocabulary word.

## Starting

1. Open the Dashboard.
2. Select the Writing card.

## Grading

Each submitted sentence is graded on two dimensions:

- Accuracy: how correct the grammar and meaning are.
- Fluency: how natural the sentence sounds.

Both are shown as percentages in the session summary alongside the count of graded sentences.

## Finishing

After a round you can Continue Practice for another set or tap Done to return to the Dashboard.
